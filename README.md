# All Sport Price Ledger

A polished inventory pricing dashboard for All Sport International.

## Live site

After GitHub Pages finishes deploying, open:

https://rafajcb787.github.io/all-sport-price-ledger/

## What it includes

- All 2,848 inventory items from the source workbook
- Search, filters, pagination, and margin health
- Editable prices with a required reason
- Per-item price history and CSV export
- All Sport International branding
- About section crediting Rafael Castrillo Beltran

The presentation edition lives in `docs/` and deploys through GitHub Actions. Because GitHub Pages is static hosting, edits are saved per visitor in browser storage rather than shared across users.
